The Path to Support

  1. Verification - Only verified organizations can request donations.
  2. Submit Request - Fill out the form with donation details, proof of need, a QR code for payment, and other required information.
  3. Admin Review - The admin reviews the request and either approves or rejects it.
  4. Listing & Donations - Approved requests are displayed on the Kindness Corner page, where donors can contribute. If a donor wishes to support a cause, they can scan the QR code displayed in the donation request for a secure transaction.
  5. User Contribution - Visitors can explore donation requests, learn about different causes, and choose to contribute. They can make a donation by scanning the provided QR code, helping organizations provide better care for animals in need.
